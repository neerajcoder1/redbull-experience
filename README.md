# Overview

Red Bull Cinematic Digital Experience is a modern interactive landing page built to showcase premium front-end engineering and motion design.

The experience combines cinematic storytelling, scroll-driven product transitions, realistic lighting, custom-rendered 3D visuals, and fluid UI interactions to deliver a luxury digital product presentation inspired by Apple's product launches.

The project demonstrates advanced animation architecture using **GSAP ScrollTrigger**, **Framer Motion**, and **Lenis**, while maintaining high performance with **Next.js App Router**.

# Features

## Cinematic Hero Experience

* Full-screen immersive hero section
* Dynamic looping energy background
* Premium dark cinematic gradient overlays
* Responsive typography and layout

---

## Scroll-Driven Storytelling

Powered by **GSAP ScrollTrigger**, every scroll interaction progresses the story through multiple product scenes.

### Product Evolution

* Classic Energy Can
* Liquid Splash & Crystal Shards
* Cryogenic Frost Explosion
* Cyber Wings Energy Portal

Each transition includes:

* Scale animations
* Opacity blending
* Layer morphing
* Smooth timeline synchronization

---

## Interactive Product Anatomy

Explore detailed product components through animated hotspots.

Highlights include:

* Pull Tab
* Double Seam Construction
* Matte Surface Finish
* Recyclability Information

Interactive hover and click animations create an engaging product exploration experience.

---

## Flavor Experience

Interactive flavor selector featuring multiple Red Bull editions.

Included variants:

* Classic
* Sugarfree
* Tropical Edition

Changing flavors dynamically updates:

* Product appearance
* Ambient lighting
* Background glow
* Transition animations

---

## Premium Cursor

Custom dual-layer cursor featuring:

* Magnetic interactions
* Smooth inertia
* Hover scaling
* Dynamic color transitions
* Interactive feedback

---

## Smooth Scrolling

Implemented using **Lenis** for:

* High-performance scrolling
* Perfect GSAP synchronization
* Fluid motion across the entire application

# Installation

### Clone the repository

```bash
git clone https://github.com/neerajcoder1/redbull-experience.git
```

### Navigate into the project

```bash
cd redbull-experience
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Visit:

```text
http://localhost:3000
```

---

# Production Build

```bash
npm run build
```

Run the production server:

```bash
npm start
```
